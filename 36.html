Linear Programming
Linear Programming & Its Applications
•
It is an important optimization (maximization or minimization) technique used in decision
making is business and everyday life for obtaining the maximum or minimum values as
required of a linear expression to satisfying certain number of given linear restrictions.
•
Involve finding maximum profit, minimum cost, or minimum use of resources etc.
•
Used in industry, commerce, management science etc.
Linear Programming Problem and its Mathematical Formulation: Definitions
•
Optimal value
o
Maximum or Minimum value of a linear function
•
Objective Function
o
The function which is to be optimized (maximized/minimized)
o
Linear function Z = ax + by, where a, b are constants, which has to be maximised or
minimized is called a linear objective function.
o
Eg:Z = 250x+ 75y where variables x and y are called
decision variables
•
Linear Constraints
o
System of linear inequations/equations under which objective function is to be optimized
o
Linear inequalities/equations or restrictions on the variables of a linear programming
problem
o
Also called as Overriding Conditions or Constraints
o
The conditions x ≥ 0, y ≥ 0 are
called non-negative restrictions.
•
Non-negative Restrictions
o
All the variables considered for making decisions assume non-negative values.
•
Optimisation problem
o
A problem which seeks to maximise or minimise a linear function (say of two variables x
and y) subject to certain constraints as determined by a set of linear inequalities
o
Linear programming problems (LPP) are special type of optimisation problems.
•
Note:
o
The term linear implies that all the mathematical relations used in the problem are linear
relations
o
The term programming refers to the method of determining a particular programme or
plan of action.
